## Краткое содержание:
Проект файлового хранилища, упрощенная копия Google Диск. Проект включает 2 модуля: server и frontend, которые являются независимыми приложениями.

### Backend:
* Java version: 8
* Netty framework
* MySQL

### Frontend:
* JavaFX
* Netty framework

### Сборка и запуск:
* Для работы приложения необходимо установить сервер MySQL и создать базу данных: file_storage_database, затем создать таблицу users в этой базе данных. Скрипт для таблицы можно взять из файла sql: backend/sql/users.sql.
* Сервер запускается файлом ServerNetty и работает на порту 8189. При запуске сервера создается папка users, которая выступает корнем хранилища файлов пользователей.
* Frontend запускается файлом ClientApp и при запуске пытается установить соединение с сервером по адресу: /localhost:8189.
* После запуска ClientApp зарегистрировать аккаунт. При успешной регистрации аккаунта на сервере, в корневой директории хранилища "users" создастся директория хранилища пользователя, название которой соответствует логину пользователя.
* Войти в приложение используя свои данные регистрации.
